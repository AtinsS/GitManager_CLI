# 🚀 Git Manager CLI

> **Git без боли. Без команд. Без лишних действий.**

**Платформа:** Windows · **Язык:** Batch Script · **Версия:** 2.0

---

## 📖 О проекте

**Git Manager CLI** — интерактивная оболочка над Git, которая превращает терминал в понятное меню.

Все репозитории — GitHub, GitLab, Bitbucket, Azure DevOps, Codeberg, Gitea — в одном месте.

```diff
- git add .
- git commit -m "fix"
- git push origin main

+ Просто выбери действие → инструмент сделает остальное
```

## 🧰 Возможности

### 📁 Управление репозиториями
- ✅ Несколько репозиториев из разных хостингов в одном окне
- ✅ Распределение проектов по группам
- ✅ Авто-определение хостинга по URL
- ✅ Просмотр статуса и текущей ветки прямо из главного меню

### ⚡ Git-операции
- `Git status` — проверка состояния
- `Git pull` — обновление
- `Git add + commit + push` — коммит с комментарием за одно действие
- `Git merge` / `merge --abort` — слияние веток и отмена
- Создание и переключение веток
- Просмотр веток для слияния

### ⏪ Откат изменений
- `Git reset` — откат на конкретный коммит (soft / mixed / hard)
- `Git revert` — отмена коммита с сохранением истории

### 🤖 Автоматизация
- **Авто-коммиты** — коммит по таймеру (каждые N минут) с опциональным push
- **Массовое обновление** — `git pull` для всех репозиториев одной кнопкой

### 🌐 Локализация
- Авто-определение языка системы
- Два комплекта скриптов: `ru/` и `en/`

## 🚀 Быстрый старт

**1.** Установи [Git](https://git-scm.com/)

**2.** Скачай проект:
```batch
git clone https://github.com/AtinsS/GitManager.git
```

**3.** Запусти `gitmanager.bat`

**4.** *(Опционально)* Запусти `install.bat` — после этого можно использовать в косоли команду `gitmanager`

## 📂 Структура проекта

<details>
<summary><b>Показать структуру</b></summary>

```
GitManager/
├── GIT-MANAGER.BAT        # Точка входа (авто-определение языка)
├── gitmanager.bat         # Обёртка для запуска из PATH

│   └── git-scripts/       # Скрипты Git-операций
└── en/
    ├── GIT-MANAGER-EN.bat # Основное меню (английский)
    └── git-scripts/       # Скрипты Git-операций
```
