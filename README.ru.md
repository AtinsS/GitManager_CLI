# 🚀 Git Manager

  

> Git без боли. Без команд. Без лишних действий.

  

**Платформа:** Windows  

**Язык:** Batch Script

  

---

  

## 🚀 Что это?

  

**Git Manager** — это интерактивная оболочка над Git, которая превращает сложный CLI в понятное меню.

Забудь про:

```

git add .

git commit -m "fix"

git push origin main

```

  

Теперь — просто выбери действие.  

**Остальное сделает инструмент.**

## ⁉️ Почему это удобно


- ⚡ **Быстрее** — рутинные операции в пару нажатий

- 🧠 **Проще** — не нужно где на каком хостинге висит этот репозиторий

- 📦 **Удобнее** — все репозитории, разных хостингов в одном месте

- 🎯 **Практичнее** — фокусируешься на коде, а не на отправке репо

## ⚙️ Быстрый старт

  

1. Установи [Git](https://git-scm.com/)

2. Скачай проект, либо склонируй репозиторий

3. Запусти: `GIT-MANAGER.BAT`

  

4. Кайфуй

### Действия внутри репозитория

  

```batch

  Репозиторий: DataInApp  Хостинг: [GitLab]  Ветка: dev
  Статус: ⚠️ есть изменения
  ════════════════════════════════════════════════════════════

  1. Git status (проверить состояние)
  2. Git pull (обновить)
  3. Git add + commit + push (с комментарием)
  4. Перейти в меню отмены изменений
  5. Просмотр истории (git log)
  6. Git merge (слияние веток)
  7. Git merge --abort (отменить слияние)
  8. Показать ветки для слияния
  9. Создать ветку
  10. Переключить ветку
  11. Авто-коммиты (каждые N минут)
  12. Вернуться в главное меню

 →

```
